Promises

They are asynchronous objects that could have three states: promise, resolved, and rejected. They are convenient when working with functions that could have any delay. In javascript, they have several subfunctions that allow you do different stuff, as the functions: then, catch, all resolved, etc.

Axios

It is a Promise-based HTTP client for the browser and node.js. This library helps you with HTTP requests.

To check string similarity, we used this npm library: string-similarity.

It finds the degree of similarity between two strings, based on Dice's Coefficient, which is mostly better than Levenshtein distance.
